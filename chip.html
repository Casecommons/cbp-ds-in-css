DS in CSS

Chips

You can style an element like a Casebook chip with the class .cb-chip. By default this will be styled like the standard chip variant.

Add a color to the chip by adding .cb-active, .cb-alert, .cb-caution, .cb-neutral-ocean, .cb-neutral-heather, or .cb-neutral-patina to its classes.
